# Numa Care E2E Tests

This repository contains end-to-end (E2E) automated tests for the Numa Care application using [Playwright](https://playwright.dev/).

## Environment
* **URL Under Test:** https://app.numaoxygen.com/
* **Framework:** Playwright + TypeScript

## How to Install
Ensure you have Node.js installed. Then run:

```bash
npm install
npx playwright install
